Hooked Web3 Provider tester

Send 1 Ether between accounts registered both in the client and in the browser. As new accounts are chosen balances will be updated.

To use this properly, make sure you have at least one account that is managed by your Ethereum client. Then, you can add new browser
accounts and pass Ether between each type of account. Underthe hood, all transactions sent from accounts handled by the browser will
be rewritten and signed by ethereumjs-accounts.

Lastly, make sure you have an Ethereum client running via RPC at http://localhost:8545 :)
